系统配置相关

系统版本：CentOS-7-x86_64
主机名: BBS
IP： 192.168.0.33 这是虚拟机自动分配
防火墙： firewall
需要关闭 SELINUX

下面列出建站需要的服务支持

  1. Apache 服务
  2. Mysql 安装
  3. Php 安装
  4. 服务配置
  5. Discuz! 安装

1、安装 Apache

[root@localhost ~]# yum install -y httpd
安装成功后查看 httpd 状态
[root@localhost ~]# systemctl status httpd.service
dead 说明服务没有启动，启动这个服务：
[root@localhost ~]# systemctl start httpd.service
Loaded-disabled不是随机启动状态，这个需要修改：
[root@localhost ~]# systemctl enable httpd.service
可以修改Listen端口，例如改为 8080，默认是 80，其实不需要修改，如果有修改配置需要重启服务。
[root@localhost ~]# systemctl restart httpd.service
此处默认端口配置。
[root@localhost ~]# vim /etc/httpd/conf/httpd.conf
测试我们的服务器是否安装完成，打开 IE，输入本机 IP 加上端口号即可访问服务器管理页面。
本机是 192.168.0.33

2、安装 Mysql

[root@localhost ~]# yum install -y MariaDB-server MariaDB-client
[root@localhost ~]# systemctl start mariadb.service
[root@localhost ~]# systemctl enable mariadb.service
下面还要设置 mysql 的密码权限，默认密码为空，此处新密码设置为 root，后面安装Discuz_X的时候会用到。
[root@localhost ~]# mysql_secure_installation

3、安装 PHP

[root@localhost ~]# yum -y install php php-bcmath php-cli php-common php-gd php-ldap php-mbstring php-mysqlnd php-pear php-pdo php-xml php-xmlrpc

重启MariaDB：
[root@localhost ~]# systemctl restart mariadb.service
重启apache：
[root@localhost ~]# systemctl restart httpd.service

5、安装 Discuz

a、这个程序系统没有集成，需要自己下载

[root@localhost ~]# wget http://download.comsenz.com/DiscuzX/3.2/Discuz_X3.2_SC_UTF8.zip
[root@localhost ~]# unzip Discuz_X3.2_SC_GBK.zip

b、将 upload 目录转移到web请求目录下

[root@localhost ~]# cp -R ./upload /var/www/html
[root@localhost ~]# cd /var/www/html
[root@localhost html]# mv upload bbs

c、进去upload给予几个目录权限访问

[root@localhost html]# chown apache:apache -R ./bbs
[root@localhost html]# cd bbs/
[root@localhost html]# chmod -R 777 config
[root@localhost html]# chmod -R 777 data
[root@localhost html]# chmod -R 777 uc_client
[root@localhost html]# chmod -R 777 uc_server

d、打开浏览器开始安装论坛

这一步网上都是说什么站点的根目录，然后域名加上install，就可以访问了，但是那些方法拷贝upload文件夹的时候，只是拷贝里面的文件，没有把整个文件夹都拷过去，所以直接加上install就可以了，但是我这个还要多一层bbs/文件夹。
这下简单了，如果自己在根目录下的一层文件夹需要添加在路径上。
http://192.168.0.33/bbs/install

但是我在这一步遇到了白屏的问题，就是安装界面怎么打开都是白屏，都来发现是vim /etc/php.ini的时候，用的是open_basedir = .:/tmp/，用这个会报错，后来我直接改成/var/www/html/bbs/:/tmp/了。
遇到白屏很麻烦，完全不知道原因，所以如果能将php产生的报错输出到屏幕就好了。
为此，我参考CentOS 下开启PHP错误提示，修改了部分配置，这样，如果遇到上面的问题，会在白屏上输出错误原因，这样就很容易发现是哪里出现的问题了。

e、开始安装论坛，点击“我同意”后，进去系统检测页面。如果一切提示OK，就继续下一步，如果提示有错误，就先去解决，一般都是上面第【c】步让你给文件权限的问题，既然上面都执行了这里一般不会报错。

但是我遇到了已经设置目录权限为777还是提示不可写的问题，在这一步一直提示不可写，都是红叉，这是由于SELinux开起的原因造成的，把SELiunx关闭就可以了
SELinux(Security-Enhanced Linux) 是美国国家安全局（NSA）对于强制访问控制的实现，是 Linux® 上最杰出的新安全子系统。NSA是在Linux社区的帮助下开发了一种访问控制体系，在这种访问控制体系的限制下，进程只能访问那些在他的任务中所需要文件。SELinux 默认安装在 Fedora 和 Red Hat Enterprise Linux 上，也可以作为其他发行版上容易安装的包得到。既然知道了原因，就很方便的可以解决。在selinux中单独开发文件权限可以自己搜索。这里我只是简单说明下怎么关闭selinux。
[root@localhost ~]# vim /etc/selinux/config
找到SELINUX 行修改成为：SELINUX=disabled:
重启系统即可。

一直下一步，下一步设置管理员密码，输入前面设置的数据库root密码，然后就成功安装了。
