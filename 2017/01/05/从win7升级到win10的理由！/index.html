从win7升级到win10的理由！

Jan 5, 2017 | Linux | Hits

最近发现win10有 “Ubuntu on Windows” 项目了，它可以让 Windows 10 系统兼容 Linux 应用程序，支持直接原生运行 Linux Bash 命令行以及数万个 Linux 应用程序！这是微软与 Ubuntu 母公司 Canonical 合作开发的一个超级隐秘的项目，这对于想要学习linux操作，但是苦于没有Linux系统的朋友来说是一个福音。

从此学习Linux的一些常用操作再也不需要在两个系统直接换来换去了。

根据微软的技术人员表示，Win10 的 Linux Bash 支持「并不是使用虚拟机/模拟器」实现的，而是嵌入一个完整的 Ubuntu Linux 环境。技术上是实时将 Linux 的系统调用转换为 Windows 的系统调用，你可以把它看做是 Wine 是一种反向技术。微软将其称为「Windows 下的 Linux 子系统」 (Windows Subsystem for Linux)。

对于熟悉 Linux 命令的朋友，以后只需在 Win10 的命令行窗口里输入bash回车即可进入 /bin/bash 的环境了，然后你就可以使用数以万计的 Ubuntu 软件库中的程序，包括熟悉的 apt-get、ssh、rsync、grep、git、vim、emacs、tar、php、mysql、python、wget、curl、gcc、diff、patch、sort、awk、sed、gpg 等等等。

详细步骤：

  1. 首先将Win10系统升级到最新的Build 14316，然后到系统设置——更新和安全——针对开发人员——选择开发者模式。
  2. 然后系统搜索“程序和功能”，选择“开启或关闭Windows功能”，开启Windows Subsystem for Linux (Beta)，并重启系统。
  3. 安装Bash，需要开启命令行模式，然后输入“bash”，第一次需要下载和安装，安装好就可以使用了。
