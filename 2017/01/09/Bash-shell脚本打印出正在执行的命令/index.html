Bash-shell脚本打印出正在执行的命令

Jan 9, 2017 | Linux | Hits

默认情况下，bash脚本不会打印执行的每个命令，除非是这个命令本身自带这个功能，而这个有时候不太方面。
首先在运行的时候有时希望能检查程序的运行进展，另外有时候后期要查看执行的具体命令是什么。
所以，将正在执行的命令打印出来是很有必要的。
如下的方法可以让bash脚本打印出执行的命令：
1、在脚本里添加-v
set -v
或者
#!bin/bash -v

2、在脚本里添加-x
set -x
或者
#!/bin/bash -x

3、在命令行里添加
$ bash -v script.sh
或者
$ bash -x script.sh

就个人而言，比较喜欢在脚本里添加-x。
因为-v仅能打印出脚本里的命令，没有用实际值等替换掉各个命令中的变量，这个在后期查看的时候还是很不方便的，而-x则很好的解决了这个问题。
